# HR Analytics Dashboard (Power BI)

This project presents an HR Analytics Dashboard built using Power BI to analyze employee data and understand workforce trends.

## Objective
To analyze employee attrition, performance, and workforce distribution to support better HR decision-making.

## Tools Used
- Power BI

## Key Insights
- Employee attrition rate analysis  
- Department-wise employee distribution  
- Gender and age group analysis  
- Job role and salary insights

## Conclusion
This dashboard helps in identifying key factors affecting employee attrition and improves decision-making for HR teams.
